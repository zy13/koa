# koa

* package.json
```
通过命令npm init生成，用于描述项目名称、版本、作者、相关依赖等信息，其中scripts字段为脚本运行的命令行定义，dependencies为项目运行所需要的依赖，devDependencies为项目开发运行所需的依赖
```

* npm i module-name --save-dev
```
安装模块，项目开发过程中使用的模块，并保存模块信息到package.json中的devDependencies
```

* npm run dev: 开启服务端

* gitignore
```
代码提交时，需要过滤掉的文件
```

* app.js
```
项目启动入口文件
```
